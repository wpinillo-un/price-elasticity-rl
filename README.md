# Setup on macOS and Linux

Run the following commands in the terminal:

```bash
python3 -m venv .venv
source .venv/bin/activate
source setup.sh
```

# Setup on Windows

Run the following commands in the terminal:

```bash
python3 -m venv .venv
.venv\Scripts\activate
setup
```

# Running Tests

Run the following command in the terminal:

```bash
pytest
```
